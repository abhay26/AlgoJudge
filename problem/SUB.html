Sub two numbers

Ali is facing some problem with substraction of two numbers. Can you please help him.

Input Specification

First line will contain N, the number of testcases.
Then will follow N pairs of numbers a,b.

Output Specification

N lines, where ith line being sub of ith pair of numbers.

Constraints

  • Easy dataset - 100 points
    • N < 1 000
    0 < a,b < 1 000

Sample Input

3
2 3
4 5
6 7

Sample Output

-1
-1
-1
